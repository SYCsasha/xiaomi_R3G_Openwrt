# xiaomi_R3G_Openwrt
这是一个利用Github action小米路由器3G的openwrt自动云固件编译

基于lean的openwrt与kanzok8的openwrt-packages

仅集成passwall(xray core)、AdGuard Home、ShadowSocksR Plus+（xray core)、SmartDNS
